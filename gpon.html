GPON Router Sozlamalari

GPON Router nima?

GPON (Gigabit Passive Optical Network) - bu yuqori tezlikdagi internet aloqasini ta'minlovchi zamonaviy texnologiya. GPON router optik tolali kabel orqali internetni qabul qilib, uni simsiz (Wi-Fi) va simli tarmoq orqali qurilmalarga uzatadi.

GPON router oddiy routerlardan farqli ravishda, optik tolali kabel orqali ishlaydi va ancha yuqori tezlikda internet aloqasini ta'minlaydi.

GPON Router qanday ishlaydi?

GPON router quyidagi asosiy qismlardan iborat:

  • Optik signal qabul qiluvchi port (SC/APC yoki SC/UPC)
  • Ethernet portlari (LAN)
  • Wi-Fi antennalari
  • Elektr ta'minoti

GPON Routerni ulash

1

Optik kabelni ulash

Provayderdan kelgan optik kabelni routerning GPON portiga ulang. Odatda bu port yashil rangda bo'ladi.

Optik kabelni ulaganda ehtiyot bo'ling - uni bukmaslik va ezmaslik kerak!
2

Elektr ta'minotini ulash

Routerning quvvat adapterini elektr manbaiga ulang. Router yonganini indikator chiroqlaridan tekshiring.

3

Internet kabelini ulash

Kompyuterni routerning LAN portlaridan biriga ulang. Bu port odatda sariq yoki ko'k rangda bo'ladi.

Admin panelga kirish

1

Brauzerga kirish

Kompyuteringizda brauzer (Chrome, Firefox va h.k.) oching va quyidagi IP manzilni kiriting:

192.168.1.1
2

Login ma'lumotlarini kiritish

Standart login ma'lumotlari:

Login: admin Parol: admin
Xavfsizlik uchun standart parolni o'zgartirishni tavsiya qilamiz!

Internet sozlamalarini kiritish

1

WAN sozlamalari

Admin panelda "Network" yoki "WAN" bo'limini toping va quyidagi sozlamalarni kiriting:

  • Connection Type: DHCP (Auto IP)
  • VLAN ID: Provayder bergan ID (odatda 100)
2

Wi-Fi sozlamalari

Wi-Fi tarmoq nomini (SSID) va parolini o'rnating:

  • Tarmoq nomi: O'zingiz xohlagan nom
  • Xavfsizlik turi: WPA2-PSK
  • Parol: Kamida 8 ta belgidan iborat murakkab parol
Barcha sozlamalarni kiritgandan so'ng "Save" yoki "Apply" tugmasini bosing va routerni qayta ishga tushiring.
